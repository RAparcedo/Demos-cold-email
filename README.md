# Demos Cold Email — Tenerife Teasers

Single-screen HTML teaser pages for cold email outreach to Tenerife businesses.

## Structure

| Folder | Sector |
|---|---|
| `tourism/` | Surf schools, boat tours, excursions, diving... |
| `tourism/output/` | Generated tourism teasers |
| `restaurant/` | Restaurants, cafés, bars, tapas... |
| `restaurant/output/` | Generated restaurant teasers |
| `real-estate/` | Property agencies, villa sales, rentals... |
| `real-estate/output/` | Generated real estate teasers |

## How to use with Claude Code

1. Open a terminal in this repo folder
2. Run `claude`
3. Tell Claude which business to generate a teaser for — it will pick the right agent automatically
4. The output HTML file will be saved in `/output/`

## Example prompts

- *"Generate a teaser for a surf school called WaveRiders, lessons from €35, 90min sessions"*
- *"Create a restaurant teaser for El Rincón del Mar, Canarian seafood, CTA is Reserve a Table"*
- *"Make a real estate teaser for Island Living Properties, tagline: Find your place in the sun"*
